# Note Taker Starter Code

## Description

---

This program is intended to be used for creating, saving, and deleting notes. The application utilizes Express.js which will save and retrieve your notes from a JSON file which serves as a database.

## Installation

---

    - Download or clone this repository,
    - Make sure to have Node.js installed,
    - Open up your integrated terminal and type "npm install" or "npm i" to install the required npm packages. (Express)
    - Nodemon is not required to run this application, it was only needed as part of the development process.

## Usage

---

Application is inititalized by opening up the integrated terminal and then typing the following command

    node server.js

After the command is run, you will see a message stating

    Note app listening at http://localhost:3001

From here, you can either copy the link and paste it into your browser, or to follow the link by (ctrl + clicking) that link within the terminal.

You can alternatively choose to use the live app that Heroku has already deployed.

[Link to Application Deployed from Heroku](https://arcane-mesa-96269.herokuapp.com/)
